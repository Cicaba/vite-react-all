# vite-react-all

- Vite2
- React
- React-Router-dom6
- Redux
- React-Redux
- redux-persist
- axios
- TypeScript

## 说明

该项目的目的是：减少React开发者在搭建项目是使用的时间。

目前该项目已经初步完成Layout组件布局。

目前主题只有一套，无法切换。

基本可以在项目中开发使用。
